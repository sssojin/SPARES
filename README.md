# SPARES

You need to install packages using `install.packages`:
- survival
- randomForestSRC
- ranger
- xgboost
- dplyr
- e1071
- parallel
- doParallel
- GeneralizedHyperbolic
- dplyr

The SPARES model consists of two modules: SPARES_Imputation and SPARES_ASP.
Please refer to the file `SPARES_run.Rmd`.
